# Counting Hypercube Nets

This project implements an algorithm to calculate the number of nets of hypercubes of dimension $n$.

It is based on the blog post:
[Calculating the number of nets of hypercubes](https://postchimpblog.wordpress.com/2022/05/29/calculating-the-number-of-nets-of-hypercubes/)

## Usage

Run with cargo:
```bash
cargo run --release
```

Run tests:
```bash
cargo test
```
